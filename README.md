The official CLI scaffolding tool to initialize new projects with the **fxd4.js** framework.

## Quick Start

You can generate a new fxd4 project instantly without manual downloads using `npx`:

```bash
npx create-fxd4 my-awesome-app
```

Once the installation is complete, navigate to your project directory and start the development server:

```bash
cd my-awesome-app
npm run dev
```
